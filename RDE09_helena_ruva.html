Incorporando multimídia ao site

Tag iframe

Usada para incorporar mídias ao documento HTML, como vídeos, áudios, sites...

A sintaxe: < iframe src=" link " height=numero width=numero >< /iframe >

Tag video

Essa tag é usada quando você quer usar um vídeo já hospedado no seu computador e não na web.
O ideal é que você coloque várias opções do mesmo vídeo no código,para caso sua primeira não funcione o código automaticamente usa a segunda. Veja a sintaxe:
< video poster="link " controls>
< source src="link 1" type="tipo de video 1">
< source scr="link 2" type="tipo de video 2">
< source src="link 3" type="tipo de video 3">
< /video >

Tag audio

Para usar áudio no seu site, você deve usar músicas e sons sem direitos autorais, para isso é possível acessar o Youtube por exemplo.
O áudio é semlhante ao vídeo, deve estar já hospedado no seu computador, também é possível disponibilizar outras opções caso a primeira não funcione no navegador. Veja a sintaxe:
< audio preload= "metadata" controls >
< source src="link audio 1" type="tipo audio 1" >
< source src="link audio 2" type="tipo audio 2" >
< source src="link audio 3" type="tipo audio 3" >
< /audio >

Tag picture

Essa tag é usada para agrupar imagens. Podemos adaptar a imagem de acordo com o tamanho da página web ou do dispositivo usado, usando o tamanho máximo em pixels. Veja a sintaxe:
< picture >
< source media="(max-width: maximo em pixels 1)" srcset="link da imagem menor" >
< source media="(max-width: maximo em pixels 2)" srcset="link da imagem media" >
< source media="(max-width: maximo em pixels 3)" srcset="link da imagem grande" >
< img src="link imagem" alt="descrição" >
< /picture >
