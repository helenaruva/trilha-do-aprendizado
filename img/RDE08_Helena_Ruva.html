Novas tags

Esse documento procura explicar o uso de algumas novas tags em HTML.

  • table

    • A tag table é usada para definir onde começam e terminam tabelas em HTML, quando os dados precisam ser organizados desta forma. Dentro desta tag podem haver outras para definir
    linhas, colunas e os dados ali inseridos.

  • del

    • < del > Representa uma parte do texto que foi excluída de um documento. Não é possível que o usuário veja essa representação no texto.

  • ins

    • Similar ao < del >, o ins (< ins >), Inserted text, representa uma parte do texto que foi inserida. Não é possível que o usuário veja essa representação no texto.

  • sub

    • Define uma parte do texto que deve ser mostrada em uma fonte abaixo das outras letras. Exemplo: H3. ( < sub > caracter < /sub >).

  • sup

    • Semelhante ao < sub > o sup define uma parte do texto que deve ser mostrada acima das outras letras. Exemplo: 4a. ( < sup > caracter < /sup >)

  • blockquote

    • Usado para fazer uma grande citação, mostrar um recuo do seu texto. ( < blockquote > < /blockquote >)Exemplo:

    Era uma vez uma princesa muito alta...

  • q

    • Indica que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo
    (< q > < /q >). Exemplo:

    Era uma vez uma princesa muito alta...

  • abbr

    • Representa uma abreviação e opcionalmente fornece uma descrição completa para ela. (Obama é presidente dos < abbr title="Estados Unidos da América">EUA< /abbr >). Exemplo:

    Obama é presidente dos EUA

  • address

    • Pode ser usado em uma variedade de contextos, como fornecer informação de contato de uma empresa ou indicar o autor de um artigo. ( < address > text < /address > ). Exemplo

    Mozilla Foundation
    1981 Landings Drive
    Building K
    Mountain View, CA 94043-0801
    USA

  • cite

    • Representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência. ( < cite > referência < /cite >). Exemplo:

    Mais informações em [ISO-0000] .

  • bdo

    • É usado para substituir a direcionalidade atual do texto. Tanto para esquerda quanto para a direita. ltr para a direita e rtl para esquerda. (< bdo dir="rtl" >Este texto ficará da direita para a esquerda.< /bdo> ) Exemplo:

    Este texto ficará da direita para a esquerda.

  • code

    • Apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código. ( < code > push() < /code >). Exemplo:

    push()

  • pre

    • É uma tag que tira a formatação do texto. Qualquer espaço criado dentro das tags < pre > e < /pre > vai aparecer no navegador como aparece no código. Exemplo:

     Qualquer espaço criado dentro das tags < pre > e < /pre > vai aparecer no navegador como aparece no código.

  • kbd

    • O elemento KBD é utilizado para representar uma entrada de teclado do usuário. Exemplo:

    Se voce não souber a resposta digite seu nome
