HTML

O HTML é usado para desenvolver páginas e documentos eletrônicos para a internet, significa Hyper Text Markup Language.
É ideal para se usar em sites criando seu corpo básico, também há outras linguagens que são direcionadas para trabalharem em conjunto como o CSS, que é usado para criar os estilos das páginas, e o Java Script, que consiste na parte lógica, trabalhando em conjunto com o HTML.

  • Tags

    • As tags definem a marcação de conteúdo. Cada tag tem sua funcionalidade específica, sem as tags o navegador exibiria um texto plano.

    Além disso, as tags podem ter atributos, que definem alguma informação extra ao navegador.

  • Marcação de títulos

    • AS tags de heading definem o título de seu texto e possuem seis níveis de importância.

  • Marcação de parágrafos

    • As tags de parágrafo definem você já sabe o que.

  • Marcação de quebra linhas

    • As tags de quebra de linhas produzem uma quebra no texto. Úteis para poemas ou endereços.
