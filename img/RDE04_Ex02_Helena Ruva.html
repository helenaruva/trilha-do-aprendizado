Imagens

Uma imagem é uma maneira de representação visual de uma pessoa, paisem ou objeto. Uma imagem digital é uma representação de uma imagem bidimensional fazendo uso de números binários codificados de modo a permitir seu armazenamento, transferência, impressão ou reprodução, e seu processamento pro meios eletrônicos.

Formatos de imagens

Formatos de imagens são padrões de codificação, compreensão e definição que determinam o tipo de arquivo gráfico, tais como: .BMP, .TIFF, .GIF, .SVG, .ICO, .JPEG e .PNG

Licenças e Repositórios de Imagens

As licenças Creative Commons são várias licenças públicas, que permitem a distribuição gratuita de uma obra protegida por direitos autorais. É extrema importância verificar quais as condições de uso de uma determinada imagem, e há diversos sites de uso como:

  • Unsplash
  • Wikimedia Commons
  • Pixabay
  • Rawpixel
  • FreePixel
  • Vintage Stock Photos

Marcação de Imagens

As tags de imagens e seus parâmetros definem as imagens e suas propriedades

As tags figure e figcaption foram definidas a partir do HTML 5.

Marcação de favicon

Determina um ícone de favoritos que aparece na barra de títulos. Sites para se adquirir esses ícones:

  1. Icon archive
  2. Favicon.cc
  3. Favicon.io

Marcação de listas

As marcações de listas definem> listas não-ordenadas,listas ordenadas e lista detalhada.

Listas não ordenadas
São usadas para listar itens, sem se preocupar com a sequência.
Listas ordenadas
São usadas para listar itens se preocupando com a sequência.
Lista detalhada
Como o nome indica, é uma lista utilizada para detalhar os itens.
